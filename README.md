# 🐂 AI A-Stock 短线分析系统

基于 Python 的 A 股短线量化分析工具集，支持 **T+1 隔日交易** 选股推荐，内置动量策略、超跌反弹、多因子评分、风险安全门等完整选股流程。

---

## 📦 环境依赖

```bash
pip install pandas numpy stockstats mootdx requests
```

| 库 | 用途 |
|---|---|
| `mootdx` (tdxpy) | 通达信行情数据（实时行情 + 历史K线） |
| `stockstats` | 技术指标计算（RSI / MACD / KDJ / BOLL） |
| `pandas` | 数据处理与计算 |
| `requests` | 腾讯行情 API（补股票名称）、同花顺强势股接口 |

## 🚀 快速开始

### 1. 运行主力动量策略

```bash
python friday_3to5_mootdx.py
```

- 全市场扫描当日涨幅 3%-7% 的股票
- 逐只计算技术指标（RSI / MACD / KDJ / BOLL / 量比 / 均线）
- 扣除安全惩罚分（乖离/缩量/连涨透支/RSI超买/高价/上影线）
- 输出风险标签 + TOP10 排行 + 通过安全门的推荐

### 2. 运行超跌反弹策略

```bash
python oversold_scan.py
```

- 全市场扫描当日大跌的股票
- 计算反弹潜力评分（下影线/BOLL位置/RSI超卖/支撑位）
- 检测连阴天数并扣分
- 输出风险标签 + 通过安全门的推荐

### 3. 精选超跌反弹（长下影+支撑）

```bash
python pick_oversold.py
```

- 聚焦当日跌幅 2%-8% 且带有长下影线的股票
- 检查 MA20/MA60 支撑位距离
- 精选放量 + 低RSI + 有支撑的标的
- 人工精选模式，偏保守

## 🛡️ 安全门机制

为防止推荐亏损股，两个主力策略均内置多重安全检查：

| 检查项 | 惩罚分 | 说明 |
|-------|--------|------|
| 乖离率 > 25% | -15 | 距离均线太远，回调风险大 |
| 乖离率 > 20% | -10 | |
| 量比 < 0.3 | -20 | 无量下跌，没有资金关注 |
| 量比 < 0.5 | -10 | 缩量严重 |
| RSI > 75 | -10 | 超买区域，回调预期 |
| RSI > 70 | -5 | |
| 5日涨幅 > 15% | -10 | 短期透支 |
| 连阳 ≥ 5天 | -8 | 获利盘积累 |
| 连阴 ≥ 4天 | -10 | 趋势恶化 |
| 股价 > ¥80 | -5 | 高价股波动大 |
| 上影线 > 3% | -5 | 抛压较大 |
| MACD 死叉 | -8~-12 | 趋势转弱 |
| KDJ-J > 100 | -5 | 超买 |

**大盘环境检测**：自动统计 500 只样本股的涨跌比，分类为 强势/偏强/震荡/偏弱/弱势 五档，弱势市场自动提高推荐阈值。

**安全门规则**：只有通过量比 ≥ 0.5、RSI ≤ 70、乖离 ≤ 20%、不连阴 4 天以上的股票才会被推荐。弱势市场（评分 ≤ -2）时直接输出 "不建议买入"。

## 📊 评分系统

### 多因子评分维度

| 因子 | 满分 | 说明 |
|------|------|------|
| 均线排列 | 25 | MA5 > MA10 > MA20 多头排列 |
| RSI 强度 | 20 | 55-70 最佳攻击区间 |
| MACD 趋势 | 15 | DIF > DEA，柱线放大 |
| 量能配合 | 15 | 量比 > 1.5 为放量 |
| 乖离率 | 10 | 3%-12% 适中 |
| KDJ 信号 | 5 | K 线黄金交叉 |
| 连阳趋势 | 5 | 3连阳以上加分 |

### 评分等级

| 总分 | 等级 | 建议 |
|------|------|------|
| ≥ 85 | 🟢 强烈推荐 | 重仓关注 |
| 70-84 | 🟢 推荐 | 正常仓位 |
| 55-69 | 🟡 关注 | 轻仓试探 |
| 40-54 | ⚪ 观望 | 暂时观察 |
| < 40 | 🔴 回避 | 不建议介入 |

## 🔬 回测验证

`backtest_score.py` 对评分系统做滚动窗口回测：

- 每只股票拉取 250 日 K 线
- 从第 60 天起，每个交易日计算截面评分
- 记录评分 vs 未来 1/3/5/10 日实际收益
- 按分数段统计胜率、平均收益、盈亏比
- 验证评分-收益 Spearman 秩相关系数
